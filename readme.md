# Lista de Compras - QuickList

---

## 🚀 Funcionalidades

- Escrever um novo item e adicioná-lo à lista.
- Selecionar itens da lista pré definida.
- Exclusão do item.
- Display mensagem "O item foi excluído da lista."
- Remoção da mensagem automática (ou manual).

---

## 🛠️ Tecnologias utilizadas

- **HTML5** → Estrutura da aplicação.
- **CSS3** → Estilização da interface.  
- **JavaScript (ES6+)** → Lógica da aplicação e manipulação do DOM.
- **Media queries** → Mobile first.

---

## 📂 Como Executar o Projeto

1. Clone este repositório: https://github.com/matheusoliveira06/Lista-de-compras.git 
2. Abra o arquivo `index.html` no navegador.
